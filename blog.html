1. What are the differnces between local storage and session storage?

Local storage and Session storage are the storages the browser gives us. Local storage is permanent and has no expire date. But Session storage is temporary.It expires with closing the tab.

2. What are the differnces between global scope and block scope?

Global scope means the variables or functions that we can access from anywhere of the code.But Block scope can not be accessed outside of its scope.But a child scope can access its parents although the parent scope can not access its childs. So here is a example like variable decalred with var keyword can be accessed from anywhere of the whole code. But variable decalred in a function with the keyword const or let can not be accessed from out of that function.

3. What is javascript event loop?

Javascript is single threaded language but it can run ascynchronusly too. In the run time js does its tasks one by one but if there is asynchronus task, js sends it to a queue and whenever its done , it comes back to the stack and gives the output.This whole complicated process which is hard to explain in writing blog, is called event loop.
